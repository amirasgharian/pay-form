# 🏦 پروژه آموزشی شبیه‌ساز انتقال وجه بانکی

این پروژه یک تمرین آموزشی نیمه‌کاره از سایت [Quera](https://quera.org) بوده که به عنوان تمرین React ارائه شده بود.  
بخش زیادی از کدها و ساختار پروژه به صورت ناقص در اختیار دانشجو قرار می‌گیرد و هدف، **تکمیل کامپوننت‌ها، مدیریت استیت‌ها و پیاده‌سازی منطق انتقال وجه** است.  
این مخزن نسخه‌ی کامل‌شده‌ی پروژه است که توسط من تکمیل شده است.

---

## 📌 ویژگی‌ها
- پیاده‌سازی چند مرحله‌ای (Multi-Step Form) برای انتقال وجه:
  - وارد کردن کارت مبدأ
  - وارد کردن کارت مقصد و مبلغ
  - نمایش گزارش نهایی انتقال
- پیاده‌سازی تایمر ۱۲۰ ثانیه‌ای برای دریافت رمز پویا (OTP)
- مدیریت و ارسال اطلاعات بین مراحل مختلف
- طراحی ساده و قابل فهم برای تمرین مفاهیم پایه

---

## 🛠️ تکنولوژی‌ها
- React.js 
- JSX
- CSS ساده
- Context & Props برای مدیریت جریان داده‌ها

---

## 📂 ساختار پوشه‌ها
src
├── components # اجزای عمومی مثل Input, BackButton, FactorRow
├── constants # ثابت‌ها مثل StepVariants
├── features # هر مرحله از فرم (SourceCredit, TargetCredit, ReportTransport)
├── layouts # لایه‌ها و چیدمان اصلی (TransportLayout)
├── App.jsx # نقطه اصلی منطق پروژه
├── index.css # استایل‌ها
└── index.js # ورودی اصلی پروژه
